# JJ Painting & Hardwares

[![Netlify Status](https://api.netlify.com/api/v1/badges/3042740d-09cd-409b-a513-1964cc064504/deploy-status)](https://app.netlify.com/projects/jjpaint/deploys)

A full-stack web application for JJ Painting & Hardwares, built with Next.js (frontend) and a Node.js backend.

## Project Structure

```
├── frontend/       # Next.js frontend application
├── backend/        # Node.js backend API
├── netlify.toml    # Netlify deployment configuration
└── package.json    # Root package scripts
```

## Deployment

This project is deployed on **Netlify**. Every push to the main branch triggers an automatic production deploy.

- **Live Site**: [jjpaint.netlify.app](https://jjpaint.netlify.app)
- **Deploy Dashboard**: [Netlify Projects](https://app.netlify.com/projects/jjpaint/deploys)
